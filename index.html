📦 Arquivos Disponíveis

Você tem 2 opções para hospedar seu relatório:

✅ OPÇÃO 1: HTML (RECOMENDADO)

Arquivo: relatorio_amorinha.html

Tamanho: ~50KB

Vantagem: Arquivo único, sem dependências externas

Melhor para: Algo rápido e sem complicações

✅ OPÇÃO 2: React

Arquivo: relatorio_amorinha.jsx

Vantagem: Código limpo e fácil de editar

Melhor para: Projetos maiores ou integração React

🚀 Como Hospedar o Site

OPÇÃO A: Abrir Localmente (Instantâneo)

Baixe o arquivo relatorio_amorinha.html
Clique duas vezes para abrir no navegador
Pronto! Site está funcionando offline

OPÇÃO B: Hospedar Gratuitamente

🌐 1. GitHub Pages (Gratuito, Fácil)

Crie uma conta em github.com
Crie um novo repositório chamado amorinha-relatorio
Faça upload do arquivo relatorio_amorinha.html
Renomeie para index.html
Vá em Settings → Pages → Branch: main
Seu site estará em: https://seu-usuario.github.io/amorinha-relatorio

☁️ 2. Netlify (Gratuito, Super Rápido)

Acesse netlify.com
Clique em "Add new site" → "Deploy manually"
Arraste e solte o arquivo relatorio_amorinha.html
Seu site estará online em segundos!

📁 3. Vercel (Gratuito)

Acesse vercel.com
Clique em "New Project"
Faça upload do arquivo HTML
Site ao vivo em um link único

🔗 4. Surge.sh (Gratuito)

Instale o surge (uma vez):

npm install -g surge

Publique seu site:

surge relatorio_amorinha.html

🎯 Nossa Recomendação

Para apresentação rápida: Use Netlify ou abra localmente
Para algo permanente: Use GitHub Pages + domínio próprio
Para total controle: Use VPS com seu domínio

🔧 Personalizações Futuras

Se quiser alterar:
  • Cores: Edite o arquivo HTML procurando por #3b82f6, #EC4899, #2D1B4E
  • Dados: Atualize os arrays horaData, campanhas, regiaoData, etc
  • Logo: Substitua o SVG no header
  • Textos: Procure e atualize os textos diretamente

📞 Dúvidas?

Se o site não funcionar:
  • Certifique-se de usar um navegador moderno (Chrome, Firefox, Safari, Edge)
  • Limpe o cache do navegador (Ctrl+Shift+Del)
  • Se hospedar, aguarde 5-10 minutos para propagação DNS
